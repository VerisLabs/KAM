CustodialAdapter

Git Source

Inherits: BaseAdapter, Initializable, UUPSUpgradeable

Adapter for custodial address integrations (CEX, CEFFU, etc.)

Simple adapter that transfers assets to custodial addresses and tracks virtual balances
